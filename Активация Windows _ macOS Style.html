Terminal — Activation Assistant

Активация Windows

Простой способ активировать систему через командную строку (KMS).

⚠️
Важно для новых ноутбуков:
Если у вас новый ноутбук с предустановленной Windows, сначала просто подключите интернет. Система должна активироваться сама официальным ключом из BIOS. Используйте этот метод только если официальная активация не сработала.
C:\>
Шаг 1

Запуск терминала

Откройте командную строку (CMD) от имени администратора:

  1. Нажмите Win + R
  2. Введите cmd
  3. Нажмите Ctrl + Shift + Enter

Вы увидите черное окно с приглашением C:\Windows\system32>

Шаг 2

Установка ключа продукта

Выберите версию вашей Windows и скопируйте соответствующую команду:

>_
Windows Home / Core
slmgr /ipk TX9XD-98N7V-6WMQ6-BX7FG-H8Q99
>_
Windows Professional
slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
Шаг 3

Активация

Введите эти две команды по очереди для завершения процесса:

>_
slmgr /skms kms.digiboy.ir
>_
slmgr /ato

После ввода последней команды подождите 10-15 секунд до появления окна "Product activated successfully".

Шаг 4

Проверка статуса

Чтобы убедиться, что активация прошла успешно, выполните эту команду:

>_
slmgr.vbs /dli

Появится окно с информацией о лицензии. Убедитесь, что статус — "Licensed".
